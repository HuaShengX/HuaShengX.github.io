安装gdb和peda插件

ubuntu自带gdb gdb -v

没有的话 sudo apt install gdb

gdb命令会显示gdb-peda$ 。

下载pwndbg(gdb插件)

下载可能会慢 换个git的源

命令行gdb会报错

Exception: Cannot override non-whitelisted built-in command “aslr”

比如peda放到pwndbg前面了，可以把顺序换下

安装gef

至此 三件套gef pwndbg peda安装好了

pwngdb和pwndbg的联合使用需要将第一行切换为source ~/pwndbg/gdbinit.py 否则会报错
